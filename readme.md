### 手枪局

#### normal

![开局站位](assets/abyss/short_normal_step1.png)

上图所示是该阵容手枪局的常规布置。omen要购置两颗烟和一个推黑sova要购置drone，a大由sova＋omen双人组做控图：具体打法是sova走到前点给drone获取a大前点站位人数，如有两人可以omen推黑，sova和omen组织双拉，目的是逼退站位。两人以上的情况详见“反控”章节。

中路开局先让omen在对局前捏一颗中路高地的悬崖烟，海神给一颗b和中路的连接烟保安跳放一颗高点摄像头看一眼信息。

钱包有摄像头可以放一颗图示位置，没有也没关系。

![正式进攻](assets/abyss/short_normal_step2.png)

上图所示是正式进攻回合。正面sova和omen逼退直架后防守方一般会交一颗防守烟，sova准备一支房梁前点箭，omen架枪或者在call提速的时候向recon照出来的二楼或者二楼下推黑。

sova准备recon的时候中路双人组碎门进中路小室。

钱包在b点单挂看信息，贴近点听脚步或者利用tp抽枪或者假装破防守道具。没信心的话可以回中路架一下可能的防守前压。

![进点混战](assets/abyss/short_normal_step3.png)

上图所示是进点要用的道具布置。正面sova和omen听指挥call双人出烟打好补枪。

中路双人组准备提速的时候先给一道图示烟墙，注意要切二楼连接边缘，碎门后海神跳拉，保安补枪。

正面双人组在碎门后进烟并进点，二楼清掉后只留保安一个在二楼小室，如果海神阵亡需要保安在二楼帮一楼双人组补枪。

钱包自己判断形式继续通b还是入侵中路。

![下包后守包](assets/abyss/short_normal_step4.png)

如图所示是守包站位，这里以4v4，海神阵亡，防守方二楼人员被清为例。omen此时两颗烟都应该转好cd了，需要omen给一颗右边箱上单向和左边箱边隔断，omen蹲矮箱架隔断烟，sova不漏左边身位晃看单向烟做好帮omen补枪的准备。

钱包此时自由度比较高，看形势选择。

保安站位比较重要，可以在小室看二楼过点，或者站一个不露给二楼警家的边沿帮一楼架枪。

#### 另外的问题

a大前点omen的关键推黑：

主要目的是逼退或者压缩防守方到包点前点方便海墙隔断。可以提出小变式：a大看到两个人不交黑也不提速，但是sova要交recon到房梁位置，中路双人组快速给烟突破二楼，和正面包夹a大，但是这个人员消耗可能要大于normal情况，具体要看防守方a大是否坚决。

面对闪光阵容的retake包点：

omen比较吃压力的是要承担打掉可击毁道具，sova来补枪，保安和钱包的位置就比较重要。保安可以让钱包拿警家信息在二楼高打低补防，钱包可以大胆一些。

包位图示位置，即使sova和omen都被换掉，钱包和保安仍然有守包能力，正常情况是omen可能被清理但是sova可以换一个，保安进场可以换一个或者换血量保活在二楼看包。

##### 一些问题

警家trap里omen的黑：

主要是拖延时间的作用，要看临场判断警家大厅人多还是人少。

omen压力比较大，因为要充当很多角色，所以在最坏的情况下omen要借助掩体箱子拖一会时间或者吸引枪线帮保安和sova争取时间。

在这些战术中保安的作用很显著，一旦正面有触发钱包和保安就会互换角色，自由人位置就到了保安身上，钱包是跟团手。

同时，保安要清楚，自己在包点队友站位没安排好的时候以保活为主；反之可以玩的更大胆。
